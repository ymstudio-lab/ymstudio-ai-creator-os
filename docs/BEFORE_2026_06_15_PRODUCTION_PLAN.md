# Production Plan Before 2026-06-15

The goal before 2026-06-15 is to create reusable assets while AI usage is available: Shorts, ebook drafts, product templates, sales copy, and sample packs.

## Core Strategy

Build once, reuse many times:

- One dashboard becomes many Shorts.
- One workflow becomes an ebook chapter.
- One app module becomes a paid template pack.
- One sample project becomes a demo, tutorial, and product example.

## Priority Assets

1. YouTube Shorts MP4 files
2. YouTube upload metadata
3. Free lead magnet
4. Paid template pack
5. Ebook outline and first chapters
6. Course lesson scripts
7. Sales page copy
8. GitHub public polish
