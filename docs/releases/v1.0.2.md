# v1.0.2 - Public Feedback MVP

Release date: 2026-05-20

## Summary

This release turns YMSTUDIO AI Creator OS from a local module bundle into a public feedback-ready MVP. It adds stronger beginner paths, clearer use cases, a richer template library, and a testable sample-project flow.

This public repo pass also removes channel/promotion planning material from the working tree and keeps the repository focused on app code, user-facing docs, public screenshots, public demo assets, tests, privacy, and security notes.

## Highlights

- Added 30 public-safe creator templates across Shorts, longform, tutorials, reviews, comparisons, thumbnails, hooks, ComfyUI, assets, and publishing.
- Reworked Template Library Korean UI copy and sample template data so the public demo renders readable text.
- Added a beginner script builder with topic, audience, length, and tone inputs.
- Added sample-project seeding across templates, script, shot plan, thumbnail, calendar, asset, and cost notes.
- Added a dedicated use cases guide for AI video beginners, Shorts creators, ComfyUI users, tutorial creators, and product/small business videos.
- Added README screenshots for dashboard, Template Library, and Script Generator.
- Added a sample-project end-to-end Playwright test.
- Added GitHub issue templates for use case suggestions and template suggestions.

## Safety Notes

- The app remains local-first and static.
- No login, backend, upload, or paid API call is required.
- Sample module data is only added to empty module storage and does not overwrite existing module data.
- JSON export/import remains the backup path.
- Public docs avoid private workflow notes, account details, credentials, and channel-specific copy.
