# YMSTUDIO AI Creator OS 사용자 가이드

## 가장 쉬운 시작

1. `index.html`을 브라우저에서 엽니다.
2. 대시보드에서 `샘플 프로젝트로 바로 시작`을 누릅니다.
3. 샘플 데이터가 각 모듈에 어떻게 연결되는지 확인합니다.
4. 내 콘텐츠에 맞게 제목, 설명, 프롬프트, 날짜, 메모를 바꿉니다.
5. 중요한 작업은 각 모듈의 `Export JSON`으로 백업합니다.

## 모듈별 사용법

### Creator Prompt Board

반복해서 쓰는 프롬프트를 저장하고 다시 찾는 공간입니다.

초보자 순서:

1. 왼쪽에서 샘플 프롬프트를 선택합니다.
2. 오른쪽에서 제목, 본문, 태그를 내 프로젝트에 맞게 수정합니다.
3. 별점과 즐겨찾기를 사용해 다시 쓸 프롬프트를 표시합니다.
4. `Copy prompt`로 복사한 뒤 ChatGPT, Claude, ComfyUI, Runway 같은 도구에 붙여 넣습니다.
5. 중요한 프롬프트 묶음은 `Export JSON`으로 보관합니다.

### Template Library

훅, 대본, 썸네일, 샷 플랜, 캘린더, ComfyUI 레시피 같은 반복 템플릿을 고르는 공간입니다.

초보자 순서:

1. `처음이면 이 3개부터` 영역에서 추천 템플릿을 확인합니다.
2. 목적에 맞는 템플릿을 선택합니다.
3. 상세 패널의 결과 설명을 읽고 필요한 템플릿인지 판단합니다.
4. 지원되는 모듈로 가져오거나 텍스트를 복사합니다.
5. 별점과 저장 기능으로 자주 쓰는 템플릿을 표시합니다.

### Script Generator

주제, 대상, 길이, 톤을 입력해 첫 대본 초안을 만드는 공간입니다.

초보자 순서:

1. Creator Project가 있다면 대시보드에서 먼저 저장합니다.
2. Script Generator에서 프로젝트 정보를 불러옵니다.
3. 훅, 장면, CTA를 수정합니다.
4. 필요한 경우 Shot Planner나 YouTube Calendar로 보냅니다.
5. 최종 대본은 JSON으로 백업합니다.

### AI Shot Planner

AI 영상의 장면과 샷을 나누고 생성 프롬프트, 도구, 상태, 자산 경로를 관리하는 공간입니다.

초보자 순서:

1. 왼쪽에서 장면을 선택합니다.
2. 가운데에서 샷을 선택합니다.
3. 오른쪽에서 설명, 프롬프트, 도구, 상태, 메모를 수정합니다.
4. 생성 도구에 넣을 프롬프트를 복사합니다.
5. 결과 파일 경로나 검토 메모를 다시 기록합니다.

### ComfyUI Workflow Manager

ComfyUI 설정, 모델 메모, 프롬프트, 실패 수정법을 프로젝트별 레시피로 저장하는 공간입니다.

초보자 순서:

1. 대시보드의 Creator Project에서 레시피를 생성합니다.
2. 모델, 해상도, steps, batch 같은 설정을 적습니다.
3. positive/negative prompt와 실패 수정법을 저장합니다.
4. 같은 스타일이 필요할 때 레시피를 다시 엽니다.
5. 레시피 JSON을 백업합니다.

### Character Consistency Tool

같은 캐릭터를 반복해서 만들기 위한 외형, 의상, 기준 이미지, 장면별 일관성 메모를 관리하는 공간입니다.

초보자 순서:

1. 캐릭터 이름과 핵심 외형을 적습니다.
2. 얼굴, 머리, 의상, 실루엣, 금지 표현을 나눠 기록합니다.
3. 기준 이미지나 파일 경로를 메모합니다.
4. 장면별로 달라지면 안 되는 요소를 확인합니다.
5. 캐릭터 바이블을 JSON으로 백업합니다.

### Thumbnail Idea Board

썸네일 훅, 한글 오버레이 문구, 레이아웃, 컬러, 이미지 프롬프트를 비교하는 공간입니다.

초보자 순서:

1. 영상 제목이나 주제를 입력합니다.
2. 짧은 오버레이 문구를 만듭니다.
3. 레이아웃과 감정을 정합니다.
4. 이미지 생성 프롬프트를 복사해 테스트합니다.
5. 선택한 썸네일 아이디어를 Calendar로 보낼 수 있습니다.

### YouTube Calendar

콘텐츠 아이디어, 제목 후보, 썸네일 메모, 제작 상태, 업로드 날짜를 로컬에서 관리하는 공간입니다.

초보자 순서:

1. 콘텐츠 제목을 입력합니다.
2. 업로드 날짜와 상태를 정합니다.
3. 제목 후보와 썸네일 프롬프트를 기록합니다.
4. 주간 계획에서 빈 날짜와 작업 밀도를 확인합니다.
5. 월간 계획은 Markdown, CSV, JSON으로 내보냅니다.

### Creator Asset Manager

이미지, 영상, 프롬프트, 파일 경로, 출처 도구, 사용권 메모를 모으는 공간입니다.

초보자 순서:

1. 자산 제목을 입력합니다.
2. 파일 경로 또는 URL을 적습니다.
3. 어떤 도구로 만들었는지 기록합니다.
4. 사용권과 외부 배포 전 확인할 사항을 메모합니다.
5. 프로젝트, 컬렉션, 상태로 정리합니다.

### API Cost Tracker

AI 도구 사용량, 크레딧, 예산, 경고 수준을 수동으로 기록하는 공간입니다.

초보자 순서:

1. 이번 달 예산과 크레딧 한도를 입력합니다.
2. 사용한 도구, 작업명, 수량, 단가를 기록합니다.
3. 월말 예상 비용과 남은 안전 예산을 확인합니다.
4. 경고가 높아지면 반복 작업이나 외부 호출을 줄입니다.
5. 월간 보고서를 Markdown 또는 CSV로 내보냅니다.

## 백업 방법

각 모듈에서 `Export JSON`을 누르면 현재 데이터가 파일로 저장됩니다. 브라우저 데이터를 지우기 전에는 반드시 JSON으로 백업하세요.

다른 브라우저나 PC에서 이어서 쓰려면 같은 모듈의 `Import JSON`으로 복구합니다.

## 데이터가 안 보일 때

- 다른 브라우저에서 열었는지 확인합니다.
- 시크릿 모드인지 확인합니다.
- 브라우저 캐시나 사이트 데이터가 삭제되었는지 확인합니다.
- 백업 JSON이 있으면 `Import JSON`으로 복구합니다.

## 입력하지 말아야 할 정보

- API 키
- 비밀번호
- 고객 개인정보
- 결제 정보
- 공개하면 안 되는 계약 자료
