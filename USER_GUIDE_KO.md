# YMSTUDIO AI Creator OS 사용법

## 가장 쉬운 시작

1. `index.html`을 브라우저로 엽니다.
2. 대시보드에서 원하는 모듈의 `모듈 열기`를 누릅니다.
3. 처음에는 샘플 데이터를 그대로 보면서 구조를 익힙니다.
4. 내 콘텐츠에 맞게 제목, 설명, 프롬프트를 바꿉니다.
5. 작업이 쌓이면 `Export JSON`으로 백업합니다.

## 모듈별 사용법

### Creator Prompt Board

프롬프트를 저장하고 재사용하는 곳입니다.

초보자 순서:
1. 왼쪽 샘플 프롬프트 선택
2. 오른쪽에서 제목/본문 수정
3. 도구 태그 선택
4. `Copy prompt`로 복사
5. Claude, ChatGPT, Runway 등에 붙여넣기

### AI Shot Planner

AI 영상 장면을 쪼개서 관리하는 곳입니다.

초보자 순서:
1. 왼쪽에서 장면 선택
2. 가운데에서 샷 선택
3. 오른쪽에서 프롬프트와 연속성 메모 수정
4. 영상 생성 도구에 프롬프트 복사
5. 결과 파일 경로를 기록

### API Cost Tracker

AI 도구 사용 비용을 기록하는 곳입니다.

초보자 순서:
1. 이번 달 예산 입력
2. 사용한 도구와 비용 입력
3. 월말 예상 비용 확인
4. 경고가 높아지면 반복 실행 줄이기

### YouTube Calendar

콘텐츠 업로드 일정을 관리하는 곳입니다.

초보자 순서:
1. 콘텐츠 제목 입력
2. 업로드 날짜 입력
3. 상태를 `idea`, `scripted`, `scheduled`처럼 이동
4. 주간 계획에서 빈 날짜 확인

### Creator Asset Manager

이미지, 영상, 프롬프트, 라이선스 메모를 모으는 곳입니다.

초보자 순서:
1. 에셋 제목 입력
2. 파일 경로 또는 URL 입력
3. 어떤 도구로 만들었는지 기록
4. 라이선스/사용 메모 작성
5. 승인됨/사용됨 상태로 정리

## 백업 방법

각 모듈에서 `Export JSON`을 누르면 현재 데이터가 파일로 저장됩니다.

브라우저 데이터를 지우기 전에는 반드시 백업하세요.

## 데이터가 안 보일 때

- 다른 브라우저로 열었는지 확인
- 시크릿 모드인지 확인
- 브라우저 캐시/사이트 데이터를 삭제했는지 확인
- 백업 JSON이 있으면 `Import JSON`으로 복구

## 입력하면 안 되는 것

- API 키
- 비밀번호
- 고객 개인정보
- 공개되면 안 되는 계약 자료
- 결제 정보
