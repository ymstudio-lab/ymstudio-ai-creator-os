## One-Line Pitch

Local-first AI Creator OS for prompts, shot planning, thumbnail ideas, API costs, YouTube scheduling, and asset management.

## Short Intro

YMSTUDIO AI Creator OS is a browser-based toolkit for AI video and content creators. It includes six local-first modules for prompt management, shot planning, thumbnail ideas, API cost tracking, YouTube scheduling, and asset organization.

It runs as static HTML/CSS/JavaScript. No login, backend, upload, or paid API call is required.

## Differentiation

YMSTUDIO AI Creator OS is not trying to replace image generators, video generators, editing apps, or project management tools.

The difference is that it sits before and around those tools as a lightweight creator operations layer.

Most AI creator workflows are scattered:

- prompts live in chat history
- shot ideas live in notes
- thumbnail concepts live in folders
- API costs live in spreadsheets
- upload schedules live in calendars
- generated assets live across random directories

YMSTUDIO AI Creator OS brings those planning pieces into one local-first workspace.

Key differences:

- Local-first: runs in the browser with no account, backend, upload, or paid API call.
- Creator-specific: built around AI video/content workflows, not generic task management.
- Modular: each tool can be used alone, but the dashboard makes them feel like one OS.
- Practical planning layer: focuses on prompts, shots, thumbnails, costs, calendars, and assets before generation/editing starts.
- Low-risk MVP: useful for testing workflows before connecting external APIs, MCP servers, or automation.
- Beginner-friendly: includes Korean UI, demo data, first-use guides, and JSON backup flows.

Positioning sentence:

> YMSTUDIO AI Creator OS is a local-first planning and operations layer for AI creators who use many generation/editing tools but need one simple place to organize the workflow around them.

## Who This Is For

The best early users are not casual AI users. They are people who already feel workflow pain because they create repeatedly.

### 1. AI Video Creators

People making YouTube videos, Shorts, TikTok clips, reels, tutorials, or AI-generated video experiments.

Why they may care:

- prompts get lost in chat history
- shot ideas are hard to reuse
- thumbnails need many variations
- upload plans become messy
- generated assets need source and license notes

Best modules for them:

- AI Shot Planner
- Thumbnail Idea Board
- YouTube Calendar
- Creator Asset Manager

### 2. Solo Creators And Small Channels

People running a channel alone or with a small team.

Why they may care:

- they do not need a heavy SaaS setup yet
- they want a simple local workspace
- they need repeatable content systems
- they want backup/export before committing to paid tools

Best modules for them:

- Creator OS Dashboard
- YouTube Calendar
- API Cost Tracker
- Creator Prompt Board

### 3. AI Tool Power Users

People using several tools such as ChatGPT, Claude, Midjourney, Runway, Kling, Pika, ComfyUI, or local models.

Why they may care:

- each tool has its own history and file outputs
- prompts and settings are hard to track across tools
- costs and credits are easy to lose track of
- reusable workflows need documentation

Best modules for them:

- Creator Prompt Board
- API Cost Tracker
- Creator Asset Manager
- Coming soon: ComfyUI Workflow Manager

### 4. Freelancers And Small Agencies

People making AI-assisted content for clients, brands, or internal marketing.

Why they may care:

- client projects need notes, assets, and usage rights
- repeatable workflows save setup time
- thumbnail and prompt variants need review
- simple local exports are useful for handoff

Best modules for them:

- Creator Asset Manager
- Thumbnail Idea Board
- AI Shot Planner
- Creator Prompt Board

### 5. Educators And Course Builders

People teaching AI video, prompt workflows, content systems, or creator operations.

Why they may care:

- the tool is easy to demo in a browser
- no login or API key is needed for students
- each module can become a lesson
- JSON exports make exercises reusable

Best modules for them:

- Creator Prompt Board
- AI Shot Planner
- Thumbnail Idea Board
- YouTube Calendar

## Who This Is Not For

This is probably not the right fit for:

- people who want AI generation inside the app
- teams that need login, permissions, and cloud sync today
- users who want full video editing
- people who only create one-off AI images
- companies handling sensitive customer data

Better positioning:

> Best for creators who already use multiple AI tools and want a simple local workspace to organize the production workflow around them.

## Longer Intro

I built YMSTUDIO AI Creator OS as a local-first workspace for AI video and content production.

The current public MVP includes:

- Creator Prompt Board
- AI Shot Planner
- API Cost Tracker
- YouTube Calendar
- Creator Asset Manager
- Thumbnail Idea Board

Everything runs in the browser and stores data in localStorage. It is not a SaaS product yet, and it does not sync data across devices. The goal is to test whether creators want a lightweight operating system for managing AI content workflows before adding heavier automation or integrations.

Feedback is welcome, especially on which module should be improved next.
