# Template Library

Local-first template library for YMSTUDIO AI Creator OS.

The first version is serverless. It includes built-in templates, local ratings, saved templates, copy, JSON backup, and direct import into supported modules.

Supported direct import targets:

- Creator Prompt Board
- Thumbnail Idea Board

Other templates are copy-only until their dedicated modules exist.
